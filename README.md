# 🎨 Donald Robertson - Art Website

Sitio web inspirado en el estilo vibrante y artístico de Donald Robertson, enfocado en mostrar obras, diseño visual y experiencia estética moderna.

---

## ✨ Características

- 🎨 Diseño moderno y minimalista
- 📱 Totalmente responsive
- 🖼️ Galería de obras con cards
- 🧑‍🎨 Sección del artista
- 🛒 Estilo tienda (precio + botón)
- 🌈 Footer llamativo con redes sociales

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
